#Features
Responsive Design: Fully optimized for desktops, tablets, and mobile devices using CSS Media Queries.
Recipe Categorization: Browse dishes by tags such as Vegetarian, Non-Veg, Desserts, and Snacks.
Detailed Recipe Pages: Each recipe includes:
Cooking Stats: Prep time, cook time, total time, and serving size.
Difficulty Levels: Color-coded tags (Easy , Medium , Hard ).
Interactive Ingredients & Instructions: Structured lists for easy following.
Nutrition Facts: Caloric and macronutrient breakdowns per serving.
Chef’s Tips: Professional advice for achieving restaurant-quality results at home.
Modern UI: Features a sleek dark navbar, hero sections, and elegant card-based layouts.

  #Technology Stack
  
HTML5: Semantic markup for accessibility and SEO.
CSS3: Custom properties, Flexbox for grid layouts, and Media Queries for responsiveness.
Typography: Georgia (serif) for headings and Arial (sans-serif) for body text to provide a classic culinary feel.
